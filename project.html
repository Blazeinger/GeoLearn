Project Description

  • What is the problem?
    • If nothing is done, Climate Change will drastically alter our world as we know it. It will end human civilization, ruin our ecosystem, and kill billions of creatures.
    The world is in an environmental crisis, and there is not enough being done to combat this crisis. Many don’t care about climate changes, don’t know enough about it,
    or have been misinformed about how our world is changing and how fast it’s changing.
    In order to ensure that we have a future ahead of us, we need to educate everyone, especially our youth. To help children, teenagers, and young adults empathize with our environment
    and have a clearer vision of the scale of change our climate is going through, we need to begin their education on a smaller, more understandable level.

  • What is our solution?
    • Our solution is a web application that automatically generates environmental science presentations for grade school teachers, ranging from elementary to high school teachers.
    These presentations will be tailored to the school’s local area, so students can better understand the environment changes around them, how these changes may be happening elsewhere,
    and the context for other changes to the environment globally

  • What is our desired outcome
    • GeoLearn wants to educate our youth with correct and interesting information to motivate them to care about the environment and pique their interest, so they may grow up to be more
    environmentally conscious, fight for more environmentally friendly practices from their governments and corporations, and possibly pursue careers in environmental sciences.

  • The initial concept for this project was provided by our sponsor in the form of Chris Doughty's Original Proposal

High Level Requirements

The Framework:

  • First start using different Google APIs
    • The Google Earth Engine API
    • The Google Google Drive Presentation API
      • We need to get these two APIs to interact with each other to better create automated presentations for teachers.
  • We need to make a GUI that is easy to interact with from its end users.
    • Must have the ability to grant admin tools to whoever is properly assigned to do so.
    • Menu options must be easy to find and easy to use for each user.
  • When a presentation is generated, it must have the most accurate information possible about the topic the user chose.
    • Must work with other applications in order to find this inforation.
      • Google Scholar.
      • Wikipedia.
      • etc.
    • Possibly have the capability to of being pre-extracted and stored in a database
  • Must have visuals for each presentaion.
    • Must have color.
      • Maps.
      • Animal Photos.
      • etc.
  • Web application must have the capability to be saved to any modern web browser.
  • Must have a basic admin interface.

Teacher Interactive Tools:

  • Must allow the user the ability to upload their own content.
    • Videos.
    • Pictures.
    • etc.
  • Must have a rating system for any uploaded content so teachers can see what is considered to be the best content.
    • Have different labels if content is from the local area and/or if the content was uploaded by a professional.
  • Users must be allowed to set age demographic for the presentations they want generated.
    • The younger the user, the more interactive it must be.

Student Interaction Tools:

  • Students must have accounts.
    • Teachers create these accounts for students.
  • Each presentations must have interactive capabilities.
    • Quizzes.
    • Questionaires.
    • etc.
  • Each form of interactive elements must have the capability to be rated by students.

Envisioned Solution

We will be creating a web application that can be accessed worldwide. This application will be used to help teachers create presentations

about environmental events (either in real-time, past events and/or future predictions) for elementary students to undersatnd. These

presentations must be engaging, through interesting information and interactive activities. The teachers can upload and rate content based

on how reliable and relevant they are to their chosen topic. Each presentation must be tailored to different grade levels so all students

can understand this content better. With this web application, we can get environmental information out easier, as well as make it interesting

for everyone that uses it.
